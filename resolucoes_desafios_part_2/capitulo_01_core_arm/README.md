# Capitulo 1 - Modulo core-only

Esta pasta contem uma resolucao objetiva para o desafio do Capitulo 1 da Parte 2.

Arquivos:

- `core_arm.h`: interface publica do modulo.
- `core_arm.c`: wrappers do core e configuracao basica do SysTick.
- `main.c`: exemplo de uso com contadores globais e secao critica.
- `startup_notes.md`: resumo conceitual sobre vector table, stack frame e EXC_RETURN.
